## 학습 목표
* 반복문을 경험한다.

## 요구사항
* 지금까지 2 ~ 5단까지 구현하기 위해 단순, 반복적인 작업이 많았다. 
* 이 같은 단순, 반복적인 작업을 변수와 반복문을 활용해 제거하면서 6단과 7단을 구현한다.

## 힌트
* while 반복문을 사용한다.
```
var i = 1;
while (i < 10) {
    // 곱셈 결과 출력
    // i 값을 1 증가
}
```

* for 반복문을 사용한다.
```
for (var i = 0; i <10; i++) {
    // 곱셈 결과 출력
}

```
